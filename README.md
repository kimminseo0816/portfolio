### 📚 Featured Projects

**[NLP — Tracking Individual Polarization and Aggression]**
Individual Study Project at UCPH (Prof. Anders Søgaard).
Analyzed 13M Reddit interactions during the 2024 US election cycle using
Sentence-BERT and weak supervision. Identified causal direction inversion
between user groups via Granger causality.

**[ML — Reddit & GameStop Meme Stock Analysis]**
Term project for 'Programming and Mathematics for Machine Learning' at SNU. 
Built end-to-end ML pipeline analyzing Reddit community activity's impact on GME stock price using 5 models
(Linear Regression, Random Forest, MLP, XGBoost, SVR).

**[Web — Spotted (Happetite)]**
